How to Get Your OpenAI API Key

**If you know me, you can type in my first name (lowercase) to skip this**
  1. Sign up or log in to your OpenAI account at platform.openai.com
  2. Go to your API Key dashboard at platform.openai.com/account/api-keys
  3. Click on "Create new secret key"
  4. Copy the API key.
  5. Paste the API key into the input field.
